Pengantar Pencarian & Penambangan Web

Web Mining atau Penambangan Web adalah proses menemukan serta mengekstrak informasi dari internet menggunakan teknik penambangan data. Informasi yang diperoleh dapat dimanfaatkan untuk pengambilan keputusan yang lebih efektif.

Tujuan

  • Memahami konsep dasar dan teknologi web mining.
  • Mampu melakukan penelitian sistem informasi dalam konteks web mining.

Jenis Web Mining

  • Web Content Mining ‚Üí mengekstrak informasi dari konten halaman web (teks, gambar, audio, video, tabel).
  • Web Usage Mining ‚Üí menganalisis pola perilaku pengguna berdasarkan data log, clickstream, dan aktivitas pengguna.
  • Web Structure Mining ‚Üí menemukan pola dari struktur hyperlink dan jaringan sosial di web.

Tantangan

  • Volume data yang sangat besar.
  • Format data beragam (HTML, XML, teks bebas).
  • Web bersifat dinamis dan kompleks.
  • Bukan domain spesifik, melainkan beragam bidang.
